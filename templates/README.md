# Competition Templates

This directory contains code templates for competitive programming.
Templates are embedded directly in the app for offline use.

## Available Templates

| Template | Language | Description |
|---|---|---|
| USACO Java | Java | Standard USACO template with fast BufferedReader I/O |
| USACO C++ | C++ | Standard USACO template with ios sync disabled |
| DP Template | Java | Dynamic programming skeleton |
| Graph Template | C++ | Dijkstra shortest path |
| Fast Scanner | Java | High-speed input reading with StringTokenizer |
| Segment Tree | C++ | Range sum segment tree |
